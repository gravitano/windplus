# Alerts

## Classes

| Class Name       | Description                                                               |
| :--------------- | :------------------------------------------------------------------------ |
| `alert`          | Alert Wrapper                                                             |
| `alert-${color}` | Alert [Color](/colors). e.g. `alert-primary`, `alert-secondary` and so on |
| `alert-dense`    | Dense Style                                                               |
| `alert-outlined` | Outlined Style                                                            |
