## Setup

**Install dependencies**:

```bash
pnpm i
```

**Run development server**:

```bash
pnpm dev
```

> Open http://localhost:3000.

**Build**:

```bash
pnpm dev
```

**Preview**:

```bash
pnpm preview
```

> Open preview at http://localhost
